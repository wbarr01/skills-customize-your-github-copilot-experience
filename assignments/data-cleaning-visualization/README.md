# 📘 Assignment: Data Cleaning and Visualization

## 🎯 Objective

Learn how to clean a raw dataset using pandas, transform data into analysis-ready form, and create visualizations that highlight key trends.

## 📝 Tasks

### 🛠️ Clean and Prepare the Dataset

#### Description
Load the provided CSV file and clean the data by handling missing values, fixing inconsistent formatting, and converting column types.

#### Requirements
Completed program should:

- Load the dataset from `data.csv` using pandas
- Detect and handle missing values in numeric and text columns
- Standardize inconsistent text values (for example, city and category names)
- Convert columns to the correct data types (e.g. dates and numeric fields)
- Add a new computed column to support analysis, such as sales per day or temperature category

### 🛠️ Create Visualizations

#### Description
Build charts that help explain the cleaned dataset and save them as image files.

#### Requirements
Completed program should:

- Create at least two different visualizations (for example, line plot, bar chart, or scatter plot)
- Save each plot as a PNG file
- Label axes, titles, and legends clearly
- Use the cleaned dataset for the visuals

### 🛠️ Summarize Key Insights

#### Description
Analyze the cleaned data and describe the most important findings.

#### Requirements
Completed program should:

- Compute summary statistics for key numeric columns
- Identify at least two trends or patterns from the data
- Print the findings in clear, student-friendly language
- Ensure the analysis is based on the cleaned data
